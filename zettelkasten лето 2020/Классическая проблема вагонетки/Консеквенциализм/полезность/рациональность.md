# рациональность
Система из научных теорий и практических техник, направленных на развитие

*   умения систематически добиваться своих целей, то есть делать оптимальный выбор, принимать [наилучшее решение](%D1%82%D0%B5%D0%BE%D1%80%D0%B8%D1%8F%20%D0%BF%D1%80%D0%B8%D0%BD%D1%8F%D1%82%D0%B8%D1%8F%20%D1%80%D0%B5%D1%88%D0%B5%D0%BD%D0%B8%D0%B9) (инструментальная р.)
*   умения создавать корректную [модель](%D0%BA%D0%B0%D1%80%D1%82%D0%B0) мира, которая систематически соответствует наблюдениям (эпистемическая р.)
